# Controller

This folder defines the shared experiment harness.

Important distinction:

- each agent has its own architecture and runtime
- the harness is the common lab environment around them

The harness should handle:

- individual conversations with each agent across 5 to 7 days
- saved notebooks, transcripts, and logs
- later group conversations between agents
