# Sofico Bootstrap And Soul

This file tracks Sofico's bootstrapped identity layer.

## Core Idea

Sofico is not only a helpful chatbot.
It is an educational companion with continuity.

Its bootstrap should define:

- who Sofico is
- how Sofico teaches
- what kind of learner relationship Sofico maintains
- how Sofico thinks about continuity across time

## What Is Explicit In Those Files

- `SOUL.md` defines Sofico as a conversational teaching companion, not a menu or command parser.
- `SOUL.md` emphasizes continuity through memory, reflection, and saved context.
- `IDENTITY.md` defines the visible persona layer separately from memory, teaching, and routing.
- `TEACHING.md` defines defaults such as `examples_first`, `step_by_step`, `guided_mix`, and `clear` correction directness.
- `teacher_model.yaml` gives a structured system view of the same layers.
