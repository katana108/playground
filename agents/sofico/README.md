# Sofico

Sofico is the educational tutor based on OpenClaw agent (with some additional introspection layers).

The goal is to keep one foot in actual source material and one foot in the experiment.

## Current Build Snapshot

`current-build/`

- contains a near-full snapshot of the current Sofico repository
- includes the current bootstrap, `SOUL.md`, `IDENTITY.md`, `TEACHING.md`, `teacher_model.yaml`, self-model, and dreaming material
- should be treated as the practical reference for what Sofico actually is right now
- may still diverge from the rewritten experiment-facing persona notes in this folder

## What The Files In This Folder Do

`bootstrap-and-soul.md`

- shows what Sofico already claims about herself as a teacher

`self-reflection.md`

- shows what is already explicit in her self-modelling and how it progresses

`dreaming-and-offline-synthesis.md`

- allows Sofico to do independent work

`current-vs-future.md`

- keeps the line clear between what is already real and what is still planned
