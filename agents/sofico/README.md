# Sofico

Sofico is the educational tutor condition in the playground.

This folder should stay close to the real Sofico project rather than inventing a separate description of it.

## Why Sofico Matters In This Comparison

- strong learner modeling
- strong pedagogical identity
- can ask calibration questions instead of only answering
- can synthesize research materials
- promising candidate for dreaming / offline integration

## What This Folder Should Capture

- links to the actual Sofico source documents
- notes on which of those documents are already real
- notes on what would need to be adapted for this experiment
- a clear split between current state and future state
