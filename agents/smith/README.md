# Smith

Smith is the baseline condition.

He is the simplest agent in the set and should stay that way.

## Purpose

- provide the control floor
- receive the same materials as the other agents
- show what happens with minimal reflective machinery

## Intended Shape

- basic assistant prompt
- same corpus access (project context)
- ordinary conversation memory if needed
- minimal saved session summaries for comparison
- no explicit self-model object
- no explicit dreaming or offline synthesis
- no strong persona beyond basic helpfulness

## Current Prompt Direction

```text
See `system_prompt.md`.
```

## Why Smith Matters

Without Smith, it becomes too easy to confuse:

- stylistic sophistication
- extra engineering
- actual developmental difference

Smith makes the rest of the comparison legible by providing the floor condition.
